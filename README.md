## Опыт работы

### Junior QA Engineer - CityAds - 2024-2025

**Выполнял задачи:**

- Функциональное, регрессионное и smoke-тестирование перед релизами.
- Тестирование REST API (Postman): валидация эндпоинтов, статус-кодов, структуры JSON.
- Анализ HTTP-запросов и ответов через Chrome DevTools.
- Проверка данных в БД (MySQL, PostgreSQL) с помощью SQL-запросов.
- Ведение дефектов в Jira, взаимодействие с разработчиками при анализе багов.

**Мои достижения:**

- Поймал критический баг в авторизации до релиза: пользователи могли получить доступ к чужим данным - исправили до того, как это увидел хоть один клиент.
- Выстроил формат общения с разработчиками по багам: ввел шаблон с шагами воспроизведения, окружением и ожидаемым результатом - переспрашивать перестали.
- За весь период работы не пропустил ни одного критического дефекта в prod.
- При горящих дедлайнах сам расставлял приоритеты: сначала критические сценарии, потом остальное - ни один релиз не ушёл без покрытия ключевых рисков.
