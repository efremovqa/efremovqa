## Опыт работы

### Junior QA Engineer - CityAds - 2024-2025

**Выполнял задачи:**

- Функциональное, регрессионное и smoke-тестирование перед релизами;
- Тестирование REST API (Postman): валидация эндпоинтов, статус-кодов, структуры JSON;
- Анализ HTTP-запросов и ответов через Chrome DevTools;
- Проверка данных в БД (DBeaver) с применением SQL-запросов;
- Ведение дефектов в Yandex Tracker, взаимодействие с разработчиками при анализе багов.

**Мои достижения:**

- Выявил критическую уязвимость в логике авторизации при тестировании edge-case сценариев, предотвратив риск утечки персональных данных.
- Внедрил стандарт ведения баг репортов (шаблоны STR, ACT, EXP, env) с обязательным вложением медиафайлов/логов, что сократило цикл Time-to-Fix и исключило этап доуточнения данных в 90% случаев.
- Обеспечил высокое качество релизов (Zero Critical Defects): за 1 год в промышленную эксплуатацию не попало ни одного критического инцидента.
- Оптимизировал процесс тестирования при дедлайнах: внедрил риск-ориентированный подход, что позволило сократить время на предрелизную проверку на 40% без потери качества ключевых функций ПО.
- Обеспечил 100% целостность данных через сквозное тестирование (E2E) на уровнях UI, API (Postman) и БД (SQL), что позволило выявить и исключить несоответствия между интерфейсом и данными в базе до релиза.
